Задача 2. Сумма знаков.

Дана сигнатура метода: public int sumLastNums (int x);
Необходимо реализовать метод таким образом, чтобы он возвращал результат
сложения двух последних знаков числах, предполагая, что знаков в числе не
менее двух. Подсказки:
int x=123%10; // х будет иметь значение 3
int у=123/10; // у будет иметь значение 12
Пример:
x=4568
результат: 14

Алгоритм решения: берём остаток деления на 10, затем исходное число делим нацело на 10, снова берём остаток деления на 10 и складываем с первым остатком

Задача 6. Тройная сумма

результат: “31 год”
Пример 3:
x=44
результат: “44 года”

Алгоритм решения: проверяем число, к какому случаю оно относится (см. в условии задачи)

Задача 10. Вывод дней недели

дней. Если в качестве строки передан не день, то выводится текст “это не день
недели”. Первый день понедельник, последний – воскресенье. Вместо if в данной

Пример 1:
x=”четверг”
результат:
четверг
пятница
суббота
воскресенье
Пример 2:
x=”чг”

это не день недели

Алгоритм решения: используем оператор switch, проверяем строку на корректное написание дня недели, выводим все дни недели, следующие за введёным (не использовать break в case)
